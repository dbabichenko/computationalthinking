This is an example of a text field - a field that accepts a single line of text as input. Note that "width" attribute specifies how many columns of text (each column has the width of one letter) will be visible without scrolling.

<input type="text" name="txtTextField" id="txtTextField" value="Default text" />

This is an example of a text area - a field that accepts multiple lines of text as input. Note that "rows" attribute specifies how many rows of text will be visible without scrolling and "cols" attribute specifies how many columns of text (each column has the width of one letter) will be visible without scrolling.

<textarea name="txtTextArea" id="txtTextArea" rows="5" cols="50">Default text</textarea>

If you want radiobuttons to be independent of each other, give their name attribues different values

This is an example of a regular HTML button. This button needs an event handler in order to perform a useful function. It DOES NOT submit the form

<input type="button" name="btnButton" id="btnButton" value="Regular HTML Button" />

This is an example of a submit button. This button will send form information to a web page specified in the ACTION attribute of the form

<input type="submit" name="btnSubmit" id="btnSubmit" value="Submit Button" />

This is an example of a password field. Password field looks like a regular text field, but when you start typing characters into it, they appear as asterisks (*).

<input type="password" name="txtPassword" id="txtPassword" />
